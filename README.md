# Shell Benchmarks

This branch hosts benchmark results for [lemtoc/dotfiles](https://github.com/lemtoc/dotfiles).

## How it works

- **`data.js`** — Historical benchmark data, updated automatically on each push to `main` by [github-action-benchmark](https://github.com/benchmark-action/github-action-benchmark)
- **`index.html`** — zsh-bench dashboard UI served at [bench.lemtoc.me](https://bench.lemtoc.me)

Do not edit `data.js` manually. The dashboard UI can be updated directly on this branch.
